# Mahjong

An implementation of [Mahjong solitaire](https://en.wikipedia.org/wiki/Mahjong_solitaire) as a Rails app.

## How to Develop

You'll need Ruby 2+ and PostgreSQL.

    bundle install
    bundle exec rake db:create db:migrate

## How to Test

    bundle install
    bundle exec rake test
